# 📖 NextChapter – Find Your Next Great Read

NextChapter is a modern web application that helps readers discover their next literary adventure. By connecting with the Open Library API, NextChapter allows users to search and explore a vast collection of books by title, author, or ISBN — making it easier than ever to turn the page to your next chapter.

Whether you're on the hunt for your next favorite novel or just browsing stories from across genres and eras, NextChapter provides a clean, intuitive interface to dive into the world of books.

---

## ✨ Features

- 🔍 Search for books by title, author, or ISBN using the Open Library API
- 📘 View detailed book information including cover, author, and description
- ❤️ Build a personalized reading list (Coming Soon)
- 🎯 Discover curated or trending reads based on user interest (Planned)
- 💅 Intuitive and mobile-friendly design with Angular & Material
- ⚙️ Clean, type-safe backend built with Spring Boot and GraphQL

---

## 🔧 Tech Stack

- **Frontend**: Angular 17, TypeScript, Angular Material
- **Backend**: Spring Boot (Java 17), GraphQL (via Spring for GraphQL)
- **Data Source**: Open Library API – [https://openlibrary.org](https://openlibrary.org)
- **Dev Tools**: Maven, Docker (optional for containerization)
